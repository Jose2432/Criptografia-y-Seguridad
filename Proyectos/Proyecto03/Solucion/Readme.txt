Para ejecutar el programa debera situarse en la carpeta donde se encuantran los archivos:
agregaPuntos.py
ecciesMain.py
ecies.py
generaxy.py
mod.py

Una vez ahi, ejecutar el comando py ecciesMain.py en la terminal, ya que ese es el Main que 
ejecutara el algoritmo con ayuda de los demas archivos .py.
